🔐 Protect PDF

Lock a PDF with a password, and/or restrict printing and copying — uses AES-256 encryption.

📤 Drag a PDF file here or click to choose

🖨️ Restrict printingStops the file from being printed
📋 Restrict copyingStops text/images from being copied or extracted
✏️ Restrict editingStops the file from being modified

ℹ️ This tool uses a new encryption engine on ApneSoftware (QPDF, open-source, Apache-2.0 licensed). Everything still runs fully in your browser — no file is uploaded anywhere.

ℹ️ About This Tool

This tool encrypts a PDF with AES-256 and a password, and can also restrict printing, copying, or editing — useful for sharing sensitive documents securely.

How to use:

  1. Upload your PDF.
  2. Set an open password (optional) and an owner password (required).
  3. Tick any restrictions you want (printing/copying/editing).
  4. Click Protect and download the locked PDF.

ℹ️ इस टूल के बारे में

यह टूल PDF को AES-256 एन्क्रिप्शन और पासवर्ड से सुरक्षित करता है, और प्रिंटिंग, कॉपी करने या एडिट करने पर भी रोक लगा सकता है — संवेदनशील डॉक्यूमेंट सुरक्षित तरीके से शेयर करने के लिए उपयोगी।

इस्तेमाल कैसे करें:

  1. अपनी PDF अपलोड करें।
  2. ओपन पासवर्ड (वैकल्पिक) और ओनर पासवर्ड (ज़रूरी) सेट करें।
  3. जो रिस्ट्रिक्शन चाहिए (प्रिंटिंग/कॉपी/एडिट) टिक करें।
  4. Protect बटन दबाएं और लॉक की हुई PDF डाउनलोड करें।
