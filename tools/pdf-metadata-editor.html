ℹ️ About This Tool

This tool lets you view and edit a PDF's hidden document properties — title, author, subject, keywords, and creator application — and save them back into the file.

How to use:

  1. Upload your PDF.
  2. The current metadata loads into the fields automatically — edit any you'd like to change.
  3. Click Save and download the updated PDF.

ℹ️ इस टूल के बारे में

यह टूल PDF की छुपी हुई डॉक्यूमेंट प्रॉपर्टीज़ — टाइटल, लेखक, विषय, कीवर्ड्स और बनाने वाला ऐप — देखने और बदलने देता है, फिर वापस फाइल में सेव कर देता है।

इस्तेमाल कैसे करें:

  1. अपनी PDF अपलोड करें।
  2. मौजूदा मेटाडेटा अपने आप फील्ड्स में आ जाएगा — जो बदलना है बदल दें।
  3. Save बटन दबाएं और अपडेटेड PDF डाउनलोड करें।
