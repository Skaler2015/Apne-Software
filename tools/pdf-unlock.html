🔓 Unlock PDF

Remove the password and restrictions from a PDF — you must know the current password.

📤 Drag a password-protected PDF here or click to choose

ℹ️ This tool uses a new encryption engine on ApneSoftware (QPDF, open-source, Apache-2.0 licensed). Everything still runs fully in your browser — no file is uploaded anywhere.

ℹ️ About This Tool

This tool removes password protection and restrictions from a PDF. You need to know the file's current password — this tool does not crack or guess unknown passwords.

How to use:

  1. Upload your password-protected PDF.
  2. Type the current password.
  3. Click Unlock and download the unprotected PDF.

ℹ️ इस टूल के बारे में

यह टूल PDF से पासवर्ड प्रोटेक्शन और रिस्ट्रिक्शन हटा देता है। इसके लिए आपको फाइल का मौजूदा पासवर्ड पता होना चाहिए — यह टूल अनजान पासवर्ड को क्रैक या अंदाज़ा नहीं लगाता।

इस्तेमाल कैसे करें:

  1. पासवर्ड-प्रोटेक्टेड PDF अपलोड करें।
  2. मौजूदा पासवर्ड डालें।
  3. Unlock बटन दबाएं और बिना प्रोटेक्शन वाली PDF डाउनलोड करें।
