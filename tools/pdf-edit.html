ℹ️ About This Tool

Add new text, highlights, or boxes anywhere on a PDF page (does not edit text already in the PDF).

How to use:

  1. Upload your PDF.
  2. Click Add Text / Add Highlight / Add Box and drag it into position.
  3. Adjust color and size, then save and download.

ℹ️ इस टूल के बारे में

यह टूल PDF पेज पर कहीं भी नया टेक्स्ट, हाईलाइट या बॉक्स जोड़ने देता है (PDF में पहले से मौजूद टेक्स्ट को एडिट नहीं करता)।

इस्तेमाल कैसे करें:

  1. अपनी PDF अपलोड करें।
  2. Add Text / Add Highlight / Add Box दबाएं और उसे मनचाही जगह खींचें।
  3. रंग और साइज़ सेट करें, फिर सेव करें और डाउनलोड करें।
