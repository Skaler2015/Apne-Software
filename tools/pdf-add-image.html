ℹ️ About This Tool

This tool lets you place an image (a photo, logo, or stamp) anywhere on any page of a PDF, then drag and resize it to fit.

How to use:

  1. Upload your PDF.
  2. Upload an image — it will appear on the current page.
  3. Drag/resize it into position, and navigate pages to add more if needed.
  4. Click Save and download the final PDF.

ℹ️ इस टूल के बारे में

यह टूल PDF के किसी भी पेज पर इमेज (फोटो, लोगो या स्टैंप) रखने देता है, फिर उसे खींचकर और साइज़ बदलकर सही जगह सेट करने देता है।

इस्तेमाल कैसे करें:

  1. अपनी PDF अपलोड करें।
  2. एक इमेज अपलोड करें — वो मौजूदा पेज पर दिख जाएगी।
  3. उसे खींचकर/साइज़ बदलकर सही जगह रखें, ज़रूरत हो तो दूसरे पेज पर जाकर और इमेज जोड़ें।
  4. Save बटन दबाएं और फाइनल PDF डाउनलोड करें।
