🔖 Bates Numbering

Stamp a continuous, sequential number (e.g. RGHS-000001) across one or more PDF documents — used for legal, government, and claim paperwork.

📤 Drag PDF files here or click to choose (multiple files allowed)

Preview of first stamp: RGHS-000001

ℹ️ About This Tool

This tool stamps a continuous sequential number across one or more PDF files — the numbering carries on from one document to the next, exactly like Bates numbering used in legal, government, and insurance/claim paperwork.

How to use:

  1. Upload one or more PDF files, in the order you want them numbered.
  2. Set a prefix/suffix, starting number, digit padding, and position.
  3. Click Apply — each file downloads separately with continuous numbering.

ℹ️ इस टूल के बारे में

यह टूल एक या कई PDF फाइलों पर लगातार चलने वाला क्रमवार नंबर लगाता है — नंबरिंग एक डॉक्यूमेंट से अगले डॉक्यूमेंट तक जारी रहती है, बिल्कुल वैसे ही जैसे लीगल, सरकारी और क्लेम के कागज़ात में Bates numbering इस्तेमाल होती है।

इस्तेमाल कैसे करें:

  1. एक या कई PDF फाइलें उसी क्रम में अपलोड करें जिस क्रम में नंबर चाहिए।
  2. प्रीफिक्स/सफिक्स, शुरुआती नंबर, डिजिट पैडिंग और पोज़िशन सेट करें।
  3. Apply बटन दबाएं — हर फाइल लगातार नंबरिंग के साथ अलग-अलग डाउनलोड होगी।
