🔒 Text Encrypt & Decrypt

Password-protect a note or message with AES-256 encryption — share the encrypted text safely, only someone with the password can read it.

ℹ️ Uses AES-256-GCM with PBKDF2 password derivation, all done locally with your browser's built-in encryption (Web Crypto API). Nothing is uploaded — and if you lose the password, the text cannot be recovered.

ℹ️ About This Tool

This tool encrypts text with a password using strong AES-256 encryption, turning it into unreadable scrambled text. Anyone with the encrypted text and the correct password can decrypt it back — useful for sharing sensitive notes over email or chat.

How to use:

  1. To encrypt: type your message, set a password, and click Encrypt.
  2. Share the encrypted text and password through separate channels for safety.
  3. To decrypt: paste the encrypted text, enter the matching password, and click Decrypt.
